Student Profiles

This project was for an programming interview. I was tasked with re-creating a given website. Each profile can be searched for by first, last, or full name. You can also add custom tags to a profile and search by those tags in addition to the profile names. The "plus" sign on the right will drop down a menu showing each profiles test score grades, which then adds to the average. This information is all being driven by an API.
