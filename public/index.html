Projects

dropDesk

dropDesk is my capstone project I created for Flatiron School. dropDesk is a ticket tracking system that implements the React-Beautiful-DnD library to create a drag and drop functionality to easily reorganize ticket sorting. The application was developed using React.js and Ruby on Rails. There is a full CRUD API built with ActiveRecord and a PostgreSQL database. The custom CSS was designed with Styled Components. You can visit the site here and you can see the source code by visiting the GitHub Repository.

Cash Prize Photo Challenge

Cash Prize Photo Challenge is a collaborative project that I worked on during the React.js phase of my Software Engineering Bootcamp at Flatiron School. Cash Prize Photo Challenge allows users to upload an image under four different challenge categories, and then users can vote on their favorite submissions. The website's frontend was programmed in React.js, while being styled using Styled Components. The backend is run on JSON Server. The application is a single-page application, formatted to route with React Router. Some technologies used in this project includes React's useEffect and useState hooks. You can visit the site here and you can see the source code by visiting the GitHub Repository.

MTG Deck Builder

MTG Deck Builder is a collaborative project that I worked on during the JavaScript phase of Flatiron School's Software Engineering Bootcamp. MTG Deck Builder allows you to search for any Magic: The Gathering card that exists. If you are unsure of the name of a card, the website will display the 20 best possible matches to your search. You can get more details of a card by clicking on it, and then you can add the card to your favorites list. Search for all the cards you want, add them to your favorites list and you can build any deck you'd like. The frontend was written entirely in vanilla JavaScript, and styled in vanilla CSS. We use the ScryFall API to display the cards you search for or add to your favorites list, and your favorites list is saved using a JSON Server. You can visit the site here and you can see the source code by visiting the GitHub Repository.
